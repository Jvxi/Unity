# Releases

## v0.2.0 - 2026-06-13

### 新增

- 新增统一二进制分析工作台，合并 PE / 虚表分析、字符串提取和 Hex 查看入口。
- 新增世界数组优先分析：识别 `GWorld`、`UWorld`、`WorldContext`、`GUObjectArray`、`GNames`、`FNamePool`、Actor/Level 等相关全局数据。
- 新增二进制分析报告中的世界数组候选、对象数组候选、名称池候选、相关字符串和证据链导出。
- 新增小说助手工作台：书库、开书问卷、灵感开书、大纲、角色、伏笔、章节写作和审查流程。
- 新增小说记忆包、RAG 检索、追读力评分、故事合同、章节提交记录、故事事件和债务追踪能力。
- 新增 OpenAI Chat Completions 兼容接口、OpenAI Responses、Anthropic Messages 等 AI 调用格式适配。

### 改进

- AI 分析提示词调整为优先分析世界数组、对象数组、名称池，再分析虚表候选。
- AI JSON 输出增加解析保护、常见 Markdown/逗号问题修复和失败后约束重试。
- 前端首页改为控制台式入口，突出二进制工作台、小说助手、聊天和设置中心。
- 前端错误展示改为读取后端业务错误，减少仅显示 `Network Error` 或 HTTP 状态码的情况。
- 二进制工作台总览指标改为自适应栅格，适配新增世界数据候选指标。
- README 更新为当前能力说明、API 概览、部署边界和配置安全说明。

### 修复

- 修复世界数组分析中字符串文件偏移与 PE RVA 不一致导致的邻近证据错位问题。
- 修复提交章节时前端未传抽取结果时高级模块可能空壳化的问题，后端会进行规则/AI 抽取补全。
- 修复 Hex 输出和部分构建细节，补充二进制工具、AI 网关、RAG、追读力和章节提交测试。

### 打包产物

- 后端开源包：`cat-tool-server-0.2.0.zip`
- Windows 桌面安装包：`cat-tool-desktop-0.2.0-windows-x64-setup.exe`
- Windows MSI 安装包：`cat-tool-desktop-0.2.0-windows-x64.msi`
- 服务器运行 JAR 由本地写入真实配置后单独构建并上传，不作为 GitHub Release 附件发布。

### 安全

- AI Key、AI 接口地址和模型选择只保存在前端本地设置，后端临时接收使用，不落库、不写日志。
- 真实数据库密码、Redis 密码、邮箱授权码、JWT 密钥、服务器地址和部署路径不得提交到 GitHub。
- GitHub Release 不上传服务器运行 JAR、带真实配置的压缩包、前端本地 `.env` 或其他私有配置。
